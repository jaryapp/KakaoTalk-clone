# 카카오톡 클론 프로젝트

Mac버전 PC카카오톡의 UI 및 기능들을 React로 구현

[데모 사이트](https://jaryapp.github.io/KakaoTalk-clone/)

## quick start
```
$ git clone https://github.com/jaryapp/KakaoTalk-clone.git
$ cd KakaoTalk-clone
$ yarn install
$ yarn start:front
```
